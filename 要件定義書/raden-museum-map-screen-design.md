## 2. トップページ（地図）

```
┌─────────────────────────────────────────┐
│ でん同士美術館マップ🐚          [報告する] │ ← ヘッダー
|─────────────┬────────────────────────────|
│                          │  美術館名      │
│   地図                   │  住所         │
│  (左60%)                 │  コラボ情報     │
│                          │  投稿一覧      │
│                          │  Google Mapリンク│
└─────────────┴─────────────────────────────┘

```

- 凡例（🔵公式コラボ／🟢リスナー訪問のみ）を地図の隅に表示する
- ピンをクリック → 美術館詳細ページへ遷移（モーダルではなくページ遷移でシンプルにする）
- 新着報告リストの項目をクリックしても該当美術館詳細へ遷移する
- 右上「報告する」ボタン → 報告投稿フォームへ

## 5. 配色・トーンの方針

明るいテーマではなく、美術館らしいシックなダーク基調にする。地図タイル自体は明るいため、周囲（ヘッダー・詳細パネル・背景）をダークにすることでメリハリを付ける。

| 用途 | 色 | Tailwindクラス例 |
|---|---|---|
| 背景（ヘッダー・詳細パネル） | チャコールグレー | `bg-neutral-900` |
| パネル内カード | ワントーン明るいグレー | `bg-neutral-800` |
| 本文テキスト | オフホワイト | `text-neutral-100` |
| 補足テキスト（日付・住所等） | ミュートグレー | `text-neutral-400` |
| 区切り線・ボーダー | 控えめなグレー | `border-neutral-700` |
| アクセントカラー（ボタン・強調・公式バッジ） | ボルドー（深い赤） | `bg-rose-900` / `text-rose-300` |
| 公式コラボピン | ゴールド系 | `#caa24a` 相当 |
| リスナー訪問のみピン | ディープグリーン | `#3f6659` 相当 |
| 地図タイル | 国土地理院 淡色地図（`pale`） | ― |

- らでんさんのテーマカラー（貝殻・水色系）はロゴやワンポイントの差し色として最小限に留め、全体はダーク基調で統一する
- 公式コラボピンと通常ピンの色分けは、色覚特性に配慮しアイコン形状も変える（丸 vs 星、等）
- ボタンやリンクなどの操作要素にのみアクセントカラー（ボルドー）を使い、情報量が多い本文エリアは彩度を抑える
